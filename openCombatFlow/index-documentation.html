Open Combat Flow Documentation

Hello, and welcome to the documentation for Open Combat Flow! I hope it is easy to navigate.

Modules

  • character: Defines the virtual Character class, and the combatHandler, which handles interactions between Characters.
    • Character: A single character
    • Combat Handler: Handles interactions between characters
  • dice: Handles dice strings

Conventions

  • Dictionary Structure Definitions: Defines how informations in dictionaries ("blocks") should be used.
  • Dice Strings: Defines how dice strings should look.
